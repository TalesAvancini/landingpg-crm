# Governance Friction Analyst (`gov-friction-analyst`)

## O que é?
Uma skill analítica que atua como o **Comitê de Ética e Arquitetura** do H.O.K Forge. Ela serve para evitar que o framework se torne uma máquina burocrática impossível de operar. 

Quando um agente de código (ex: `spec-driver`) começa a reclamar das "Travas" do projeto e propõe criar scripts imensos para resolver problemas pequenos, essa skill entra em ação para analisar a dor real e encontrar a solução mais preguiçosa e eficiente possível (ex: apenas adicionar uma linha no `RULES.md`).

## Quando Usar?
Sempre que um executor devolver um relatório de bloqueios, falhas do Gatekeeper, "Mosca na Janela", ou atritos de governança.

## Como Usar?
1. Copie o relatório de dor do executor.
2. Diga ao orquestrador: `"analise as dores do executor: [COLE O RELATÓRIO AQUI]"`
3. A skill automaticamente carregará a "Constituição" do projeto (9 arquivos essenciais), filtrará as ideias estúpidas e devolverá um plano atômico de correção.
